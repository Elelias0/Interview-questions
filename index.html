AI・Python 技術ぎじゅつ質問しつもん

Preguntas técnicas de IA y Python (probable en esta vacante)
Q21
AIと機械きかい学習がくしゅうの違ちがいは何なんですか。
¿Cuál es la diferencia entre IA y machine learning?
回答例
AIは広ひろい言葉ことばで、コンピューターが人間にんげんのように考かんがえたり判断はんだんしたりする技術ぎじゅつ全体ぜんたいを指さすと理解りかいしています。機械きかい学習がくしゅうはその中なかの一つの方法ほうほうで、データからパターンを学まなんで予測よそくや判断はんだんをする技術ぎじゅつだと思おもいます。
Entiendo que la IA es un término amplio que se refiere a toda la tecnología para que las computadoras piensen o decidan como los humanos. El machine learning es uno de esos métodos, una tecnología que aprende patrones de los datos para predecir o decidir.
Q22
教師きょうしあり学習がくしゅうと教師きょうしなし学習がくしゅうの違ちがいを教おしえてください。
Explique la diferencia entre aprendizaje supervisado y no supervisado.
回答例
教師きょうしあり学習がくしゅうは、正解せいかいのラベルがついたデータを使つかって学習がくしゅうする方法ほうほうです。教師きょうしなし学習がくしゅうは、正解せいかいのラベルがなく、データの中なかのパターンやグループを見みつける方法ほうほうだと理解りかいしています。
El aprendizaje supervisado es un método que usa datos con etiquetas de la respuesta correcta. El no supervisado no tiene esas etiquetas y busca patrones o grupos dentro de los datos.
Q23
過学習かがくしゅうとは何なんですか。
¿Qué es el sobreajuste (overfitting)?
回答例
過学習かがくしゅうは、学習がくしゅうデータには強つよいですが、新あたらしいデータにはうまく対応たいおうできない状態じょうたいだと理解りかいしています。学習がくしゅうデータの細こまかい部分ぶぶんまで覚おぼえすぎることが原因げんいんだと思おもいます。
Entiendo que el sobreajuste es un estado donde el modelo funciona bien con los datos de entrenamiento pero no responde bien con datos nuevos. Creo que la causa es que memoriza demasiado los detalles específicos de los datos de entrenamiento.

Q27
RAGについて知しっていますか。
¿Conoce el RAG (Retrieval-Augmented Generation)?
回答例
はい。RAGは、AIが答こたえる前まえに、関連かんれんするデータを検索けんさくして、その情報じょうほうをもとに回答かいとうを作つくる仕組しくみだと理解りかいしています。Kitsune Studyでは、pgvectorを使つかって、文法ぶんぽうデータベースからRAGで検索けんさくし、AIチューターの回答かいとうの精度せいどを上あげる機能きのうを実装じっそうしました。
Sí. Entiendo que el RAG es un mecanismo donde la IA busca datos relacionados antes de responder, y genera la respuesta basándose en esa información. En Kitsune Study implementé, usando pgvector, una búsqueda RAG sobre una base de datos de gramática para mejorar la precisión de las respuestas del tutor de IA.
Q28

回答例
Kitsune Studyでは、生成せいせいした内容ないようをキャッシュして、同おなじ内容ないようを何度なんども生成せいせいしないようにしています。また、スピードが必要ひつような部分ぶぶんと、バッチで大量たいりょうに生成せいせいする部分ぶぶんで、違ちがうAIモデルを使つかい分わけて、コストを抑おさえる工夫くふうをしています。
En Kitsune Study, guardo en caché el contenido generado para no volver a generar lo mismo varias veces. Además, uso diferentes modelos de IA según si necesito velocidad o generación masiva por lotes, para mantener los costos bajos.

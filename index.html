日本にほんとAIへの転向てんこう

Transición hacia Japón y la IA
Q15
Kitsune Studyについて教おしえてください。
Cuénteme sobre Kitsune Study.
回答例
Kitsune Studyは、日本語にほんご学習がくしゅう者しゃ向むけのウェブアプリです。JLPT対策たいさく、語彙ごい、漢字かんじ、文法ぶんぽうの学習がくしゅう、聴解ちょうかい、読解どっかいなどの機能きのうがあります。バックエンドはFastAPIとPostgreSQL、フロントエンドはReactとTypeScriptを使つかっています。Gemini APIを使つかってAIが問題もんだいを作つくる機能きのうも実装じっそうしました。妻つまと二人ふたりで企画きかくし、私わたしは主おもに開発かいはつを担当たんとうしています。
Kitsune Study es una app web para estudiantes de japonés. Tiene preparación para el JLPT, aprendizaje de vocabulario, kanji, gramática, comprensión auditiva y lectura. El backend usa FastAPI y PostgreSQL, y el frontend React y TypeScript. También implementé una función con la API de Gemini para que la IA genere ejercicios. Lo planeamos entre mi esposa y yo, y yo me encargo principalmente del desarrollo.
